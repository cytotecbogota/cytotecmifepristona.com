# Preguntas Frecuentes

## ¿Qué es la Mifepristona?

La Mifepristona es un medicamento utilizado para interrumpir el embarazo en sus primeras etapas. Actúa bloqueando la hormona necesaria para que el embarazo continúe.

## ¿Qué es Cytotec?

Cytotec (Misoprostol) es un medicamento que se utiliza para provocar contracciones uterinas, ayudando a completar el proceso de interrupción del embarazo.

## ¿Es seguro usar estos medicamentos?

Cuando se utilizan correctamente y siguiendo las indicaciones adecuadas, estos medicamentos son considerados seguros. Es importante contar con información confiable y acompañamiento durante el proceso.
